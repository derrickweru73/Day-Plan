# Day Craft

## Project Overview
 
A minimalist, single-page web dashboard designed to capture, process, and retain basic user profile metrics while offering daily inspiration.

---

## Problems it solves
 Repetitive Data Entry: Users do not have to type their profiles over and over again every time they visit. The application remembers them automatically.
 Cluttered Information: Converts plain static data into custom, responsive dashboard visualizations, making tracking easy.
 Generic User Experiences: Eliminates boring "one-size-fits-all" pages by tailoring the interface look, layout colors, and title greetings to the specific user's life stage.

## Features
*Persistent Storage: Saves user profile data natively in the browser so details remain loaded even after closing the tab.
*Dynamic Age Conversion: Automatically transforms standard calendar years into total life months via JavaScript.
*Conditional Personalization: Adapts greetings and theme colors based on whether the user is an adult or a minor.
*Automated Mantra Engine: Uses code loops to display critical motivational reminders multiple times across the workspace.
